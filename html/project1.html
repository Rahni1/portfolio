MERN Stack E-commerce Shop

Project Features

  • Admin Dashboard: View Orders (update status of orders & view details), Manage Products (update and delete products), Create Products and Create Categories
  • Filter products by categories and price range
  • Search by product name and category
  • Email sent to customer and admin when order is made using SendGrid
  • Signup/Signin (hashed password using crypto)
  • User Dashboard: Update profile and view purchase history
  • Related Products listed on individual product page
  • Stock count - in stock/out of stock badge on each product
  • Credit Card and PayPal payment using Braintree
